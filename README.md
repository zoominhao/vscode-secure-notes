# VSCode 加密笔记插件

一个支持富文本编辑和 AES 加密存储的安全笔记插件。

## ✨ 功能特性

- 📝 **富文本编辑器** - 支持 Markdown 格式化
- 🔒 **AES 加密** - 保护隐私内容
- 💾 **自动保存** - 每30秒自动保存
- 🗂️ **笔记管理** - 侧边栏树状视图
- ⌨️ **快捷键** - 快速操作

## 🚀 快速开始

### 安装依赖

```bash
cd vscode-secure-notes
npm install
```

### 编译插件

```bash
npm run compile
```

### 调试运行

1. 在 VSCode 中打开此项目
2. 按 `F5` 启动调试
3. 在新窗口中测试插件

### 打包发布

```bash
npm install -g vsce
vsce package
```

会生成 `.vsix` 文件，可以手动安装或发布到市场。

## 📖 使用说明

### 1. 创建笔记

- 点击侧边栏"加密笔记"图标
- 点击 `+` 按钮创建新笔记
- 输入标题和内容
- 可选择是否加密

### 2. 加密笔记

- 首次使用加密功能，先设置密码：
  - 命令面板 (`Cmd+Shift+P`)
  - 搜索"设置加密密码"
- 创建/编辑笔记时勾选"🔒 加密存储"

### 3. 编辑笔记

- 点击笔记项打开编辑器
- 使用工具栏快速格式化
- 支持 Markdown 语法
- `Cmd+S` 保存

### 4. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd+Shift+N` | 创建新笔记 |
| `Cmd+S` | 保存笔记 |

## 🔧 技术栈

- **语言**: TypeScript
- **框架**: VSCode Extension API
- **加密**: crypto-js (AES-256)
- **存储**: JSON 文件
- **编辑器**: 自定义 Webview

## 🔐 安全说明

- 使用 AES-256 加密算法
- 密码仅存储在内存中
- 未加密的笔记以明文存储
- 加密笔记即使文件被盗也无法读取
